## Learnings

### Day 1
**Problem:** Missing Number (LeetCode 268)

**Key Idea:**
- Calculate the expected sum from 0 to n using:
  `n * (n + 1) / 2`
- Find the actual sum of the array.
- Missing Number = Expected Sum − Actual Sum.

**Complexity:**
- Time: O(n)
- Space: O(1)
