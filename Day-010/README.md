# Day 10 – Count Odd Numbers in an Interval Range

## Problem Details

* Problem Number: 1523
* Problem Name: Count Odd Numbers in an Interval Range
* Difficulty: Easy
* Platform: LeetCode

## Problem Statement

Given two non-negative integers `low` and `high`, return the count of odd numbers between `low` and `high` (inclusive).

### Examples

Input:
low = 3, high = 7

Output:
3

Input:
low = 8, high = 10

Output:
1

## Approach Used

Instead of iterating through every number in the range, I used a mathematical observation to directly determine the count of odd numbers.

By calculating the total numbers in the interval and analyzing whether the range length is odd or even, the answer can be obtained in constant time.

## Complexity Analysis

* Time Complexity: O(1)
* Space Complexity: O(1)
