# Day 3 – Perfect Number

## Problem Details

* Problem Number: 507
* Problem Name: Perfect Number
* Difficulty: Easy
* Platform: LeetCode

## Approach Used

To determine whether a number is a Perfect Number, I calculated the sum of all positive divisors excluding the number itself.

If the sum of these divisors equals the original number, then the number is a Perfect Number.

Example:
28 = 1 + 2 + 4 + 7 + 14

Since the sum equals 28, it is a Perfect Number.

## Complexity Analysis

* Time Complexity: O(n)
* Space Complexity: O(1)
